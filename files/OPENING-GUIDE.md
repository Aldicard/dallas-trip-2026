# Dallas Trip Planner — How to Open the File

The trip planner is a single HTML file. To get the interactive map, tabs, and voting to work, **you must open it in a web browser** (Safari, Chrome, Firefox, or Edge). If the file opens in a preview/reader tool instead, the map will be black, the tabs won't work, and voting won't function.

Follow the steps for your device below.

---

## 💻 Desktop — Windows & Mac

This is the easiest platform. On most computers, double-clicking the file is all you need.

### Received as an Email Attachment

1. Open the email and **download** the attachment (`dallas-trip.html`) to your computer.
2. Find the downloaded file (usually in your **Downloads** folder).
3. **Double-click** the file — it will open in your default browser automatically.

> If it opens in a text editor or the wrong app, right-click the file → **Open with** → choose **Chrome**, **Firefox**, **Safari** (Mac), or **Edge**.

### Received via iMessage on Mac

1. Click the file in the conversation to download it.
2. Find it in your **Downloads** folder.
3. Double-click to open in your default browser.

---

## 📱 iPhone & iPad

iPhone is the trickiest platform. When you tap an HTML attachment, iOS opens it in **Quick Look** — a read-only preview that does not run JavaScript. The map will be black and nothing interactive will work. You need to redirect it to **Safari**.

### Received via Mail (Apple Mail App)

1. Tap the attachment in the email to open the Quick Look preview.
2. Tap the **Share icon** (box with arrow pointing up) in the bottom-left corner.
3. Scroll through the share sheet and tap **Open in Safari**.

> Don't see "Open in Safari"? Scroll further right in the share sheet or tap **More** to find it.

### Received via iMessage / SMS

1. Tap and **hold** the file attachment in the message.
2. Tap **Share** from the menu that appears.
3. Tap **Open in Safari** from the share sheet.

### Received via Gmail App

1. Tap the attachment to download it.
2. Tap **Open in...** when prompted, and select **Safari**.
3. If it opens in Gmail's in-app browser instead, tap the **three dots (⋮)** in the top-right corner and select **Open in Safari**.

### Saved to Files / iCloud Drive

1. Open the **Files** app and find `dallas-trip.html`.
2. Tap and **hold** the file (do not just tap — that opens Quick Look).
3. Tap **Share** from the menu.
4. Tap **Open in Safari**.

---

## 🤖 Android

Android is generally more straightforward — Chrome usually opens HTML files directly.

### Received via Gmail App

1. Tap the attachment to download it.
2. Once downloaded, tap **Open** — it will open in **Chrome** automatically.
3. If it opens in a document viewer instead, tap the **three dots (⋮)** → **Open with** → **Chrome**.

### Received via Google Messages / SMS

1. Tap the attachment to download it.
2. Tap **Open** and select **Chrome** if prompted to choose an app.

### Saved to Files / Downloads

1. Open your **Files** app (or **My Files** on Samsung).
2. Navigate to **Downloads** and find `dallas-trip.html`.
3. Tap the file and select **Chrome** when asked which app to use.

> On some Android versions you may see a warning like "This file type can harm your device." This is a generic browser warning for HTML files — tap **OK** to proceed.

---

## ✅ How to Tell It's Working

Once the file is open correctly in a browser, you should see:

- A **dark-themed page** with a gold "Dallas Trip" header
- **Five tabs** across the top: Map, Airbnbs & Cost, Restaurants, Shopping, Gyms
- An **interactive map** with colorful pins (not a black rectangle)
- Ability to tap pins and see popups

If you still see a black map or can't tap the tabs, close the tab and try the steps above again — the most common fix is making sure it opened in Safari (iPhone) or Chrome (Android) rather than a preview or reader app.
